┌──────────────────────────────────────────────────────────────────────────────┐
│                              🚀 QUICK START                                  │
└──────────────────────────────────────────────────────────────────────────────┘

    STEP 1: Get Gemini API Key
    └─ Visit: https://makersuite.google.com/app/apikey

    STEP 2: Setup (Choose One)
    
    Option A - Automated:
    └─ Run: .\setup.ps1
    
    Option B - Manual:
    ├─ Backend:
    │  ├─ cd backend
    │  ├─ python -m venv venv
    │  ├─ venv\Scripts\activate
    │  ├─ pip install -r requirements.txt
    │  └─ copy .env.example .env (add API key)
    │
    └─ Frontend:
       ├─ cd frontend
       └─ npm install

    STEP 3: Run Application
    
    Terminal 1 (Backend):
    ├─ cd backend
    ├─ venv\Scripts\activate
    └─ python app.py
    
    Terminal 2 (Frontend):
    ├─ cd frontend
    └─ npm run dev

    STEP 4: Open Browser
    └─ http://localhost:3000

│                              🔧 TROUBLESHOOTING                              │
└──────────────────────────────────────────────────────────────────────────────┘

    ⚠️ PowerShell Script Error
    └─ Fix: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

    ⚠️ Port Already in Use
    └─ Fix: Change port in app.py or vite.config.js

    ⚠️ Module Not Found
    └─ Fix: Reinstall dependencies (pip install / npm install)

    ⚠️ Gemini API Error
    └─ Fix: Check API key in .env, restart server

    ⚠️ CORS Error
    └─ Fix: Ensure backend on port 5000, check flask-cors
